关于我们

传统工艺与现代技术的完美结合

老字号腊肉工厂创立于1985年，至今已有近40年的历史。我们始终坚持"质量第一，顾客至上"的经营理念，传承百年腊味制作工艺。

工厂占地5000平方米，拥有现代化的生产车间和严格的质量管理体系。我们的产品选用当地优质猪肉，经过精心挑选、科学配方、传统腌制和自然风干等多道工序制作而成。

多年来，我们不断改进工艺，在保留传统风味的同时，引入现代食品加工技术，确保产品安全卫生、品质稳定。

产品销往全国各地，并深受消费者喜爱。我们期待与您分享这份来自传统的美味！

GitHub Pages部署指南

1. 创建GitHub仓库

登录GitHub后，点击右上角的"+"图标，选择"New repository"

仓库名称：输入您的仓库名（如：lawei-factory）

选择 Public（公开仓库）

勾选"Initialize this repository with a README"

点击"Create repository"按钮

2. 上传网站文件

在仓库页面，点击"Add file" > "Upload files"

将本HTML文件拖到上传区域或点击"choose your files"

点击"Commit changes"按钮

3. 启用GitHub Pages

进入仓库的"Settings"选项卡

在左侧菜单中选择"Pages"

在"Source"部分选择"main"分支和"/ (root)"文件夹

点击"Save"按钮

4. 访问您的网站

等待几分钟后，刷新页面

您会看到一个绿色提示框，显示您的网站URL：

https://[您的用户名].github.io/[仓库名]

点击该链接即可访问您的网站
